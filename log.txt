=== Step 1: Refactor NeuralService to Multi-Modality ===
Date: 2026-05-25

File modified:
  backend/app/services/neural.py  (53 → 106 lines)

What changed:
  - NeuralService._engine (single BaseModelEngine) → _engines (dict[str, BaseModelEngine])
  - NeuralService.last_latency (simple float) → _latency dict + @property delegate
  - NeuralService.last_fidelity (simple float) → _fidelity dict + @property delegate

New multi-modality API:
  register_engine(modality, engine)   — register engine without loading
  load_engine(modality)               — load weights for a modality
  unload_engine(modality)             — unload and release resources
  is_loaded(modality)                 — check if engine is registered + loaded
  get_latency(modality)               — per-modality latency (ms)
  get_fidelity(modality)              — per-modality confidence
  all_modalities (property)           — list of registered modality names
  infer(data, modality="gesture")     — run inference on specified engine

Backward-compat preserved (delegate to "gesture"):
  model_loaded (property)            → is_loaded("gesture")
  last_latency (property)            → _latency["gesture"]
  last_fidelity (property)           → _fidelity["gesture"]
  load_model()                       → auto-registers + loads GestureModelEngine
  unload_model()                     → unload_engine("gesture")
  infer(landmarks) [old signature]   → infer(data, "gesture")

Callers verified (no changes needed in):
  backend/app/services/gesture.py   — .model_loaded, .infer(landmarks)
  backend/app/routes/status.py      — .model_loaded, .last_latency, .last_fidelity
  backend/app/routes/model.py       — .load_model(), .unload_model()
  backend/app/main.py               — .load_model(), .unload_model()

Tests passed:
  - Import: OK
  - load_model() → model_loaded: True, all_modalities: ['gesture']
  - is_loaded("gesture"): True, is_loaded("speech"): False
  - infer(data) [default]: OK, infer(data, "gesture"): OK
  - last_latency/last_fidelity properties: OK
  - get_latency("gesture") / get_fidelity("gesture"): OK
